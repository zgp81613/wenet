Add <sos> and <eos> labels.

  • eos (int) – index of <eeos>

  • ignore_id (int) – index of padding

Returns:

(B, Lmax + 1) ys_out (torch.Tensor) : (B, Lmax + 1)

Return type:

ys_in (torch.Tensor)

Examples

>>> sos_id = 10
>>> eos_id = 11
>>> ignore_id = -1
>>> ys_pad
tensor([[ 1,  2,  3,  4,  5],
        [ 4,  5,  6, -1, -1],
        [ 7,  8,  9, -1, -1]], dtype=torch.int32)
>>> ys_in,ys_out=add_sos_eos(ys_pad, sos_id , eos_id, ignore_id)

        [10,  7,  8,  9, 11, 11]])
>>> ys_out
tensor([[ 1,  2,  3,  4,  5, 11],
        [ 4,  5,  6, 11, -1, -1],
        [ 7,  8,  9, 11, -1, -1]])
