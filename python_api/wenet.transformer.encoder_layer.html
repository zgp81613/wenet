  • mask (torch.Tensor) – Mask tensor for the input (#batch, time，time), (0, 0, 0) means fake mask.

  • att_cache (torch.Tensor) – Cache tensor of the KEY & VALUE (#batch=1, head, cache_t1, d_k * 2), head * d_k == size.

  • cnn_cache (torch.Tensor) – Convolution cache in conformer layer (#batch=1, size, cache_t2)
